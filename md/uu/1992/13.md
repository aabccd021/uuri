# [Menimbang](http://example.org/legal/document/uu/1992/13/menimbang)

* [a.](http://example.org/legal/document/uu/1992/13/menimbang/point/a) bahwa transportasi mempuny ai peranan penting dan strategis untuk memantapkan perwujudan wawasan nusantara, memperkukuh ketahanan nasional, dan mempererat hubungan antar bangsa dalam usaha mencapai tujuan nasional berdasarkan Pancasila dan Undang-Undang Dasar 1945;
* [b.](http://example.org/legal/document/uu/1992/13/menimbang/point/b) bahwa perkeretaapian sebagai sa lah satu moda transportasi tidak dapat dipisahkan dari moda-moda transportasi lain yang ditata dalam sistem transportasi nasional, mempunyai karakteristik pengangkutan secara masal dan keunggulan tersendiri, perlu lebih dikembangkan potensinya dan ditingkatkan peranannya sebagai penghubung wilayah baik nasional maupun internasional, sebagai penunjang, pendorong, dan penggerak pembangunan nasional demi peningkatan kesejahteraan rakyat;
* [c.](http://example.org/legal/document/uu/1992/13/menimbang/point/c) bahwa peraturan perundang-undangan yang mengatur perkeretaapian yang ada pada saat ini tidak sesuai lagi dengan kebutuhan dan perkembangan zaman, ilmu pengetahuan dan teknologi;
* [d.](http://example.org/legal/document/uu/1992/13/menimbang/point/d) bahwa untuk meningkatkan pembinaan dan penyelenggaraan perkeretaapian sesuai dengan perkembangan kehidupan rakyat dan PRESIDEN REPUBLIK INDONESIA - 2 - bangsa Indonesia serta agar lebih berhasilguna dan berdayaguna dipandang perlu menetapkan ketentuan mengenai perkeretaapian dalam Undang-undang;
# [Mengingat](http://example.org/legal/document/uu/1992/13/mengingat)
Pasal 5 ayat (1), Pasal 20 ayat (1), dan Pasal 33 Undang-Undang Dasar 1945; Dengan persetujuan DEWAN PERWAKILAN RAKYAT REPUBLIK INDONESIA MEMUTUSKAN: Menetapkan : UNDANG-UNDANG TENTANG PERKERETAAPIAN.

# [BAB I: KETENTUAN UMUM](http://example.org/legal/document/uu/1992/13/bab/0001)

### [Pasal 1](http://example.org/legal/document/uu/1992/13/pasal/0001)
Dalam Undang-undang ini yang dimaksud dengan:
* [1.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0001) Perkeretaapian adalah segala sesuatu yang berkaitan dengan sarana, prasarana, dan fasilitas penunjang kereta api untuk penyelenggaraan angkutan kereta api yang disusun dalam satu sistem;
* [2.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0002) Kereta api adalah kendaraan deng an tenaga gerak, baik berjalan sendiri maupun dirangkaikan dengan kendaraan lainnya, yang akan ataupun sedang bergerak di jalan rel,
* [3.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0003) Jaringan jalur kereta api adalah seluruh jalur kereta api yang terkait satu sama lain yang menghubungkan berbagai tempat sehingga merupakan satu sistem; PRESIDEN REPUBLIK INDONESIA - 3 -
* [4.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0004) Jalur kereta api khusus adalah jalur kereta api yang digunakan secara khusus oleh badan usaha tertentu untuk menunjang kegiatannya;
* [5.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0005) Fasilitas keselamatan perkeretaapian adalah perangkat bangunan, peralatan, dan perlengkapan yang digunakan untuk menunjang kelancaran dan keselamatan perjalanan kereta api;
* [6.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0006) Sarana kereta api adalah segala sesuatu yang dapat bergerak di atas jalan rel;
* [7.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0007) Prasarana kereta api adalah jalu r dan stasiun kereta api termasuk fasilitas yang diperlukan agar sarana kereta api dapat dioperasikan;
* [8.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0008) Fasilitas penunjang kereta api adalah segala sesuatu yang melengkapi penyelenggaraan angkutan kereta api yang dapat memberikan kemudahan serta kenyamanan bagi pengguna jasa kereta api;
* [9.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0009) Pengguna jasa adalah setiap orang dan/atau badan hukum yang menggunakan jasa angkutan kereta api baik untuk angkutan orang maupun barang;
* [10.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0010) Badan penyelenggara adalah badan usaha milik negara yang melaksanakan penyelenggaraan angkutan kereta api;
* [11.](http://example.org/legal/document/uu/1992/13/pasal/0001/version/19920511/point/0011) Menteri adalah Menteri yang bertanggung jawab di bidang perkeretaapian. PRESIDEN REPUBLIK INDONESIA - 4 -
 


# [BAB II: ASAS DAN TUJUAN](http://example.org/legal/document/uu/1992/13/bab/0002)

### [Pasal 2](http://example.org/legal/document/uu/1992/13/pasal/0002)
Perkeretaapian sebagai salah satu moda transportasi nasional diselenggarakan berdasarkan asas manfaat, adil dan merata, keseimbangan, kepentingan umum, keterpaduan, dan percaya pada diri sendiri.


### [Pasal 3](http://example.org/legal/document/uu/1992/13/pasal/0003)
Perkeretaapian diselenggarakan dengan tujuan untuk memperlancar perpindahan orang dan/atau barang secara masal, menunjang pemerataan, pertumbuhan dan stabilitas serta sebagai pendorong dan penggerak pembangunan nasional.
 


# [BAB III: PEMBINAAN](http://example.org/legal/document/uu/1992/13/bab/0003)

### [Pasal 4](http://example.org/legal/document/uu/1992/13/pasal/0004)
Perkeretaapian dikuasai oleh Negara dan pembinaannya dilakukan oleh Pemerintah.


### [Pasal 5](http://example.org/legal/document/uu/1992/13/pasal/0005)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0005/version/19920511/ayat/0001)
Pembinaan perkeretaapian diarahkan untuk meningkatkan peranserta angkutan kereta api dalam keseluruhan moda transportasi secara terpadu.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0005/version/19920511/ayat/0002)
Ketentuan mengenai pembinaan perkeretaapian sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0005/version/19920511/ayat/0001), diatur lebih lanjut dengan Peraturan PRESIDEN REPUBLIK INDONESIA - 5 - Pemerintah.
 


# [BAB IV: PENYELENGGARAAN](http://example.org/legal/document/uu/1992/13/bab/0004)

### [Pasal 6](http://example.org/legal/document/uu/1992/13/pasal/0006)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0006/version/19920511/ayat/0001)
Perkeretaapian diselenggarakan oleh Pemerintah dan pelaksanaannya diserahkan kepada badan penyelenggara yang dibentuk untuk itu berdasarkan peraturan perundang-undangan yang berlaku.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0006/version/19920511/ayat/0002)
Badan usaha lain selain badan penyelenggara sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0006/version/19920511/ayat/0001), dapat diikutsertakan dalam kegiatan perkeretaapian atas dasar kerjasama dengan badan penyelenggara.

#### [Ayat (3)](http://example.org/legal/document/uu/1992/13/pasal/0006/version/19920511/ayat/0003)
Bentuk dan syarat-syarat kerjasama sebagaimana dimaksud dalam [ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0006/version/19920511/ayat/0002), diatur lebih lanjut dengan Peraturan Pemerintah.


### [Pasal 7](http://example.org/legal/document/uu/1992/13/pasal/0007)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0007/version/19920511/ayat/0001)
Untuk menunjang kegiatan badan usaha di bidang industri, pertanian, pertambangan, dan kepariwisataan oleh badan usaha yang bersangkutan dapat digunakan kereta api khusus.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0007/version/19920511/ayat/0002)
Ketentuan sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0007/version/19920511/ayat/0001), diatur lebih lanjut dengan Peraturan Pemerintah. PRESIDEN REPUBLIK INDONESIA - 6 -
 


# [BAB V: PRASARANA DAN SARANA](http://example.org/legal/document/uu/1992/13/bab/0005)

### [Pasal 8](http://example.org/legal/document/uu/1992/13/pasal/0008)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0008/version/19920511/ayat/0001)
Pemerintah menyediakan dan merawat prasarana kereta api.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0008/version/19920511/ayat/0002)
Penyediaan dan perawatan prasarana sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0008/version/19920511/ayat/0001), dapat dilimpahkan kepada badan penyelenggara.

#### [Ayat (3)](http://example.org/legal/document/uu/1992/13/pasal/0008/version/19920511/ayat/0003)
Pengusahaan prasarana sebaga imana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0008/version/19920511/ayat/0001), dilaksanakan oleh badan penyelenggara.


### [Pasal 9](http://example.org/legal/document/uu/1992/13/pasal/0009)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0009/version/19920511/ayat/0001)
Badan penyelenggara menyediakan dan merawat sarana kereta api.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0009/version/19920511/ayat/0002)
Penyediaan dan perawatan sarana sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0009/version/19920511/ayat/0001), dapat dilakukan oleh badan usaha lain sebagaimana dimaksud dalam [Pasal 6 ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0009/version/19920511/ayat/0002), dengan cara kerjasama dengan badan penyelenggara.

#### [Ayat (3)](http://example.org/legal/document/uu/1992/13/pasal/0009/version/19920511/ayat/0003)
Pengusahaan sarana sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0009/version/19920511/ayat/0001) dan [ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0009/version/19920511/ayat/0002), dilaksanakan oleh badan penyelenggara.


### [Pasal 10](http://example.org/legal/document/uu/1992/13/pasal/0010)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0010/version/19920511/ayat/0001)
Prasarana dan sarana kereta api yang dioperasikan wajib mempunyai keandalan dan memenuhi persyaratan keselamatan.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0010/version/19920511/ayat/0002)
Untuk memenuhi ketentuan sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0010/version/19920511/ayat/0001), terhadap setiap prasarana dan sarana kereta api dilakukan pemeriksaan dan pengujian. PRESIDEN REPUBLIK INDONESIA - 7 -

#### [Ayat (3)](http://example.org/legal/document/uu/1992/13/pasal/0010/version/19920511/ayat/0003)
Syarat keselamatan dan tata cara pemeriksaan serta pengujian diatur lebih lanjut dengan Peraturan Pemerintah.


### [Pasal 11](http://example.org/legal/document/uu/1992/13/pasal/0011)
Pemerintah mengembangkan rancang bangun dan rekayasa perkeretaapian.


### [Pasal 12](http://example.org/legal/document/uu/1992/13/pasal/0012)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0012/version/19920511/ayat/0001)
Pengoperasian prasarana dan sarana kereta api hanya dapat dilakukan oleh tenaga-tenaga yang telah memenuhi kualifikasi keahlian.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0012/version/19920511/ayat/0002)
Persyaratan keahlian dan tata cara mendapatkan kualifikasi sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0012/version/19920511/ayat/0001), diatur lebih lanjut dengan Peraturan Pemerintah.


### [Pasal 13](http://example.org/legal/document/uu/1992/13/pasal/0013)
Untuk kelancaran dan keselamatan pengoperasian kereta api, Pemerintah menetapkan pengaturan mengenai jalur kereta api yang meliputi daerah manfaat jalan, daerah milik jalan, dan daerah pengawasan jalan termasuk bagian bawahnya serta ruang bebas di atasnya.


### [Pasal 14](http://example.org/legal/document/uu/1992/13/pasal/0014)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0014/version/19920511/ayat/0001)
Dilarang membangun gedung, membuat tembok, pagar, tanggul dan bangunan lainnya, menanam je nis pohon yang tinggi serta menempatkan barang pada jalur kereta api baik yang dapat mengganggu pandangan bebas, maupun dapat membahayakan keselamatan kereta api. PRESIDEN REPUBLIK INDONESIA - 8 -

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0014/version/19920511/ayat/0002)
Ketentuan mengenai larangan sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0014/version/19920511/ayat/0001), diatur lebih lanjut dengan Peraturan Pemerintah.


### [Pasal 15](http://example.org/legal/document/uu/1992/13/pasal/0015)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0015/version/19920511/ayat/0001)
Perlintasan antara jalur kereta api dengan jalan dibuat dengan prinsip tidak sebidang.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0015/version/19920511/ayat/0002)
Pengecualian terhadap prinsip sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0015/version/19920511/ayat/0001), hanya dimungkinkan deng an tetap mempertimbangkan keselamatan dan kelancaran, baik perjalanan kereta api maupun lalu lintas di jalan.

#### [Ayat (3)](http://example.org/legal/document/uu/1992/13/pasal/0015/version/19920511/ayat/0003)
Ketentuan mengenai perpotongan dan pengecualiannya sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0015/version/19920511/ayat/0001) dan [ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0015/version/19920511/ayat/0002), diatur lebih lanjut dengan Peraturan Pemerintah.


### [Pasal 16](http://example.org/legal/document/uu/1992/13/pasal/0016)
Dalam hal terjadi perpotongan jalu r kereta api dengan jalan yang digunakan untuk lalu lintas umum atau lalu lintas khusus, pemakai jalan wajib mendahulukan perjalanan kereta api.


### [Pasal 17](http://example.org/legal/document/uu/1992/13/pasal/0017)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0017/version/19920511/ayat/0001)
Pembangunan jalan, jalur kereta api khusus, terusan, saluran air dan/atau prasarana lain yang menimbulkan atau memerlukan persambungan, pemotongan atau penyinggungan dengan jalur kereta api, dilaksanakan dengan cara yang tidak membahayakan keselamatan perjalanan kereta api.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0017/version/19920511/ayat/0002)
Ketentuan mengenai pelaksanaan pembangunan sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0017/version/19920511/ayat/0001), diatur lebih lanjut dengan Peraturan PRESIDEN REPUBLIK INDONESIA - 9 - Pemerintah.


### [Pasal 18](http://example.org/legal/document/uu/1992/13/pasal/0018)
Badan penyelenggara sebagaimana dimaksud dalam Pasal 6 ayat (1), berwenang melarang siapapun:
* [a.](http://example.org/legal/document/uu/1992/13/pasal/0018/version/19920511/point/a) berada di daerah manfaat jalan kereta api;
* [b.](http://example.org/legal/document/uu/1992/13/pasal/0018/version/19920511/point/b) menyeret barang di atas atau melintasi jalur kereta api;
* [c.](http://example.org/legal/document/uu/1992/13/pasal/0018/version/19920511/point/c) menggunakan jalur kereta api untuk kepentingan lain, selain untuk angkutan kereta api;
* [d.](http://example.org/legal/document/uu/1992/13/pasal/0018/version/19920511/point/d) berada di luar tempat yang disediakan untuk angkutan penumpang dan/atau barang;
* [e.](http://example.org/legal/document/uu/1992/13/pasal/0018/version/19920511/point/e) mengganggu ketertiban dan/atau pelayanan umum.


### [Pasal 19](http://example.org/legal/document/uu/1992/13/pasal/0019)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0019/version/19920511/ayat/0001)
Stasiun merupakan tempat kereta api berangkat dan berhenti untuk melayani naik dan turunnya penumpang dan/atau bongkar muat barang dan/atau untuk keperluan operasi kereta api.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0019/version/19920511/ayat/0002)
Kecuali dalam hal-hal tertentu yang ditetapkan oleh badan penyelenggara, naik turunnya penumpang dan/atau bongkar muat barang hanya dapat dilakukan di stasiun.


### [Pasal 20](http://example.org/legal/document/uu/1992/13/pasal/0020)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0020/version/19920511/ayat/0001)
Selain berfungsi sebagai temp at naik atau turunnya penumpang dan/atau bongkar muat barang, di stasiun dapat dilakukan kegiatan usaha penunjang angkutan kereta api.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0020/version/19920511/ayat/0002)
Ketentuan sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0020/version/19920511/ayat/0001), diatur lebih PRESIDEN REPUBLIK INDONESIA - 10 - lanjut oleh Menteri.
 


# [BAB VI: JARINGAN PELAYANAN ANGKUTAN KERETA API](http://example.org/legal/document/uu/1992/13/bab/0006)

### [Pasal 21](http://example.org/legal/document/uu/1992/13/pasal/0021)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0021/version/19920511/ayat/0001)
Jaringan pelayanan angkutan kereta api diselenggarakan secara terpadu dalam satu kesatuan yang merupakan bagian tidak terpisahkan dari sistem transportasi secara keseluruhan.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0021/version/19920511/ayat/0002)
Jaringan pelayanan angkutan kereta api disusun dalam jaringan pelayanan angkutan antar kota dan jaringan pelayanan angkutan kota.


### [Pasal 22](http://example.org/legal/document/uu/1992/13/pasal/0022)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0022/version/19920511/ayat/0001)
Jaringan pelayanan angkutan kereta api antar kota sebagaimana dimaksud dalam [Pasal 21 ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0022/version/19920511/ayat/0002) yang berfungsi sebagai pelayanan lintas utama, melayani angkutan jarak jauh dan sedang.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0022/version/19920511/ayat/0002)
Jaringan pelayanan angkutan kereta api antar kota sebagaimana dimaksud dalam [Pasal 21 ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0022/version/19920511/ayat/0002) yang berfungsi sebagai pelayanan lintas cabang, melayani angkutan jarak sedang dan dekat.


### [Pasal 23](http://example.org/legal/document/uu/1992/13/pasal/0023)
Jaringan pelayanan angkutan kota sebagaimana dimaksud dalam [Pasal 21 ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0023/version/19920511/ayat/0002), berfungsi sebagai pelayanan lintas utama dalam satu sistem angkutan kota. PRESIDEN REPUBLIK INDONESIA - 11 -


### [Pasal 24](http://example.org/legal/document/uu/1992/13/pasal/0024)
Angkutan kereta api khusus berfungsi untuk melayani kegiatan badan usaha tertentu di bidang industri, pertanian, pertambangan, dan kepariwisataan.
 


# [BAB VII: ANGKUTAN](http://example.org/legal/document/uu/1992/13/bab/0007)

### [Pasal 25](http://example.org/legal/document/uu/1992/13/pasal/0025)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0025/version/19920511/ayat/0001)
Penyelenggaraan pelayanan angkutan orang atau barang dilakukan setelah dipenuhinya syarat-syarat umum angkutan yang ditetapkan badan penyelenggara berdasarkan Undang-undang ini.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0025/version/19920511/ayat/0002)
Karcis penumpang atau surat angkutan barang merupakan tanda bukti terjadinya perjanjian angkutan.


### [Pasal 26](http://example.org/legal/document/uu/1992/13/pasal/0026)
Penumpang dan/atau barang yang telah memenuhi syarat-syarat umum angkutan sebagaimana dimaksud dalam [Pasal 25](http://example.org/legal/document/uu/1992/13/pasal/0025), wajib diangkut oleh badan penyelenggara.


### [Pasal 27](http://example.org/legal/document/uu/1992/13/pasal/0027)
Jika terjadi pembatalan pemberangk atan perjalanan kereta api oleh badan penyelenggara, badan peny elenggara wajib mengembalikan jumlah biaya yang telah dibayar ol eh penumpang dan/atau pengirim barang.


### [Pasal 28](http://example.org/legal/document/uu/1992/13/pasal/0028)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0028/version/19920511/ayat/0001)
Badan penyelenggara bertanggung jawab atas kerugian yang PRESIDEN REPUBLIK INDONESIA - 12 - diderita oleh pengguna jasa dan/ atau pihak ketiga yang timbul dari penyelenggaraan pelayanan angkutan kereta api.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0028/version/19920511/ayat/0002)
Tanggung jawab sebagaimana di maksud dalam ayat (1), diberikan dengan ketentuan:

* [b.](http://example.org/legal/document/uu/1992/13/pasal/0028/version/19920511/ayat/0002/point/b) besarnya ganti rugi dibatasi sejumlah maksimum asuransi yang ditutup oleh badan penyelenggara dalam hal penyelenggaraan kegiatannya.


### [Pasal 29](http://example.org/legal/document/uu/1992/13/pasal/0029)
Badan penyelenggara diberi wewenang untuk:
* [a.](http://example.org/legal/document/uu/1992/13/pasal/0029/version/19920511/point/a) melaksanakan pemeriksaan terhadap pemenuhan syarat-syarat umum angkutan bagi penumpang dan/atau barang;
* [b.](http://example.org/legal/document/uu/1992/13/pasal/0029/version/19920511/point/b) melaksanakan penindakan atas pelanggaran terhadap syarat-syarat umum angkutan tersebut huruf a;
* [c.](http://example.org/legal/document/uu/1992/13/pasal/0029/version/19920511/point/c) membatalkan perjalanan kereta api apabila dianggap dapat membahayakan ketertiban dan kepentingan umum;
* [d.](http://example.org/legal/document/uu/1992/13/pasal/0029/version/19920511/point/d) menertibkan penumpang kereta api atau masyarakat yang mengganggu perjalanan kereta api.


### [Pasal 30](http://example.org/legal/document/uu/1992/13/pasal/0030)
Struktur dan golongan tarif angkutan kereta api ditetapkan oleh Pemerintah. PRESIDEN REPUBLIK INDONESIA - 13 -


### [Pasal 31](http://example.org/legal/document/uu/1992/13/pasal/0031)
Tanggung jawab sebagaimana dimaksud dalam [Pasal 28](http://example.org/legal/document/uu/1992/13/pasal/0028), dimulai sejak diangkutnya penumpang dan/atau diterimanya barang dan berakhir di tempat tujuan yang disepakati.


### [Pasal 32](http://example.org/legal/document/uu/1992/13/pasal/0032)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0032/version/19920511/ayat/0001)
Pengirim dan/atau penerima barang yang tidak mengambil barangnya dari tempat penyim panan yang ditetapkan badan penyelenggara dalam jangka waktu yang ditetapkan sesuai dengan syarat-syarat umum angkutan, dikenakan biaya pcnyimpanan barang.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0032/version/19920511/ayat/0002)
Pengirim dan/atau penerima barang hanya dapat mengambil barang setelah biaya sebagaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0032/version/19920511/ayat/0001) dilunasi.

#### [Ayat (3)](http://example.org/legal/document/uu/1992/13/pasal/0032/version/19920511/ayat/0003)
Barang yang tidak diambil seba gaimana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0032/version/19920511/ayat/0001) lebih dari waktu tertentu, dinyatakan sebagai barang tak bertuan dan dapat dijual secara lelang sesuai ketentuan peraturan perundang-undangan yang berlaku.


### [Pasal 33](http://example.org/legal/document/uu/1992/13/pasal/0033)
Pengangkutan barang berbahaya dilaksanakan sesuai peraturan perundang-undangan yang berlaku.


### [Pasal 34](http://example.org/legal/document/uu/1992/13/pasal/0034)
Badan penyelenggara wajib mengasuransikan tanggung jawabnya sebagaimana dimaksud dalam [Pasal 28](http://example.org/legal/document/uu/1992/13/pasal/0028). PRESIDEN REPUBLIK INDONESIA - 14 -

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0035/version/19920511/ayat/0002)

# [BAB VIII: PENYIDIKAN](http://example.org/legal/document/uu/1992/13/bab/0008)

### [Pasal 36](http://example.org/legal/document/uu/1992/13/pasal/0036)

#### [Ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0036/version/19920511/ayat/0001)
Selain Pejabat Polisi Negara Republik Indonesia, Pegawai Negeri Sipil tertentu di lingkungan departemen yang lingkup tugas dan tanggung jawabnya meliputi pembinaan di bidang perkeretaapian, dapat diberi wewenang khusus sebagai penyidik sebagaimana dimaksud dalam Undang-undang Nomor 8 Tahun 1981 tentang Hukum Acara Pidana, untuk melakukan penyidikan tindak pidana di bidang perkeretaapian.

#### [Ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0036/version/19920511/ayat/0002)
Penyidik sebagaimana dimaksud dalam ayat (1), berwenang untuk:
* [a.](http://example.org/legal/document/uu/1992/13/pasal/0036/version/19920511/ayat/0002/point/a) melakukan pemeriksaan atas kebenaran laporan, pengaduan atau keterangan tentang adanya tindak pidana;
* [b.](http://example.org/legal/document/uu/1992/13/pasal/0036/version/19920511/ayat/0002/point/b) memanggil dan memeriksa saksi dan/atau tersangka;
* [c.](http://example.org/legal/document/uu/1992/13/pasal/0036/version/19920511/ayat/0002/point/c) melakukan penggeledahan, penyegelan dan/atau penyitaan alat-alat yang digunakan untuk melakukan tindak pidana;
* [d.](http://example.org/legal/document/uu/1992/13/pasal/0036/version/19920511/ayat/0002/point/d) melakukan pemeriksaan tempat yang diduga digunakan untuk melakukan tindak pidana; c. meminta keterangan kepada saksi-saksi dan mengumpulkan PRESIDEN REPUBLIK INDONESIA - 15 - barang bukti dari orang dan/atau badan hukum sehubungan dengan tindak pidana; f. membuat dan menandatangani berita acara pemeriksaan; g. menghentikan penyidikan ap abila tidak terdapat cukup bukti tentang adanya tindak pidana.

#### [Ayat (3)](http://example.org/legal/document/uu/1992/13/pasal/0036/version/19920511/ayat/0003)
Pelaksanaan penyidikan sebaga imana dimaksud dalam [ayat (1)](http://example.org/legal/document/uu/1992/13/pasal/0036/version/19920511/ayat/0001) dan [ayat (2)](http://example.org/legal/document/uu/1992/13/pasal/0036/version/19920511/ayat/0002), dilakukan sesuai dengan peraturan perundang-undangan yang berlaku.
 


# [BAB IX: KETENTUAN PIDANA](http://example.org/legal/document/uu/1992/13/bab/0009)

### [Pasal 37](http://example.org/legal/document/uu/1992/13/pasal/0037)
Barangsiapa membangun gedung, membuat tembok, pagar tanggul dan bangunan lainnya, menanam jenis pohon yang tinggi serta menempatkan barang pada jalur kereta api, baik yang dapat mengganggu pandangan bebas maupun yang dapat membahayakan keselamatan kereta api sebagaimana dimaksud dalam [Pasal 14](http://example.org/legal/document/uu/1992/13/pasal/0014), dipidana dengan pidana kurungan paling lama 3 (tiga) bulan atau denda setinggi-tingginya Rp 3.000.000,- (tiga juta rupiah) dan diwajibkan membayar ganti rugi kepada badan penyelenggara serta wajib membongkar ataupun menghilangkan gangguan dimaksud.


### [Pasal 38](http://example.org/legal/document/uu/1992/13/pasal/0038)
Barangsiapa karena perbuatannya mengakibatkan rusaknya pintu perlintasan kereta api atau tanpa hak membuka pintu perlintasan kereta api pada waktu kereta api akan dan/atau sedang berjalan dipidana dengan pidana kurungan paling lama 6 (enam) bulan atau denda setinggi-tingginya Rp 6.000.000,- (enam juta rupiah) dan diwajibkan membayar ganti rugi kepada badan penyelenggara. PRESIDEN REPUBLIK INDONESIA - 16 -


### [Pasal 39](http://example.org/legal/document/uu/1992/13/pasal/0039)
Barangsiapa melakukan perbuatan yang mengakibatkan terjadinya pergeseran tanah di jalur kereta api, sebagaimana dimaksud dalam [Pasal 17](http://example.org/legal/document/uu/1992/13/pasal/0017) sehingga mengganggu atau membahayakan perjalanan kereta api, dipidana dengan pidana kurungan paling lama 3 (tiga) bulan atau denda setinggi-tingginya Rp 3.000.000,- (tiga juta rupiah) dan diwajibkan membayar ganti rugi kepada badan penyelenggara.


### [Pasal 40](http://example.org/legal/document/uu/1992/13/pasal/0040)
Barangsiapa melakukan perbuatan yang mengakibatkan rusaknya, mengurangi nilai atau tidak dapat berfungsinya atau tidak dapat berfungsi secara sempurna sarana dan/atau prasarana kereta api, dipidana dengan pidana kurungan paling lama 3 (tiga) bulan atau denda setinggi-tingginya Rp 3.000.000,- (tiga juta rupiah) dan diwajibkan membayar ganti rugi kepada badan penyelenggara.


### [Pasal 41](http://example.org/legal/document/uu/1992/13/pasal/0041)

### [Pasal 42](http://example.org/legal/document/uu/1992/13/pasal/0042)
Apabila perbuatan sebagaimana dimaksud dalam [Pasal 41](http://example.org/legal/document/uu/1992/13/pasal/0041) menyebabkan matinya orang, luka berat atau cacat dapat dipidana sesuai dengan ketentuan dalam hukum pidana. PRESIDEN REPUBLIK INDONESIA - 17 -
